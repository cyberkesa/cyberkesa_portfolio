# Настройка изображения для IdCard

## Проблема
HEIC формат не поддерживается браузерами (кроме Safari) и файл не попадает на Vercel, так как находится в `.gitignore`.

## Решение

### Вариант 1: Конвертация в JPG/PNG (Рекомендуется)

1. Откройте `IMG_6620.HEIC` в Preview (Mac) или другом редакторе
2. Экспортируйте как JPG или PNG
3. Сохраните в `public/images/id-card.jpg` (или `.png`)
4. Обновите путь в `components/sections/about-section.tsx`:
   ```tsx
   <IdCard imageSrc="/images/id-card.jpg" />
   ```

### Вариант 2: Использовать существующий файл

Если файл уже есть на сервере, просто обновите путь в `about-section.tsx`.

## Текущий статус

- Компонент `IdCard` теперь имеет fallback placeholder `[ID]` если изображение не загружается
- Обработка ошибок улучшена
- Поддержка как HEIC (через `<img>`), так и обычных форматов (через Next.js Image)

## Рекомендации

- Используйте JPG для фотографий (лучшее сжатие)
- Используйте PNG для изображений с прозрачностью
- Оптимизируйте размер файла (рекомендуется < 500KB)
- Используйте Next.js Image для автоматической оптимизации
